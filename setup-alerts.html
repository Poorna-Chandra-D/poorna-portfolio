Portfolio Visitor Alerts Setup

Get real-time notifications when visitors come to your portfolio

Telegram Setup (Recommended)

‚úÖ Best for: Instant mobile notifications, fast & reliable

Step 1: Create a Telegram Bot

1
Open Telegram and search for "@BotFather"

Tap on BotFather (official Telegram bot)

2
Send command: /newbot

Follow the prompts to create your bot

Bot name example: "Poorna's Portfolio Alerts"

Bot username example: "poorna_portfolio_bot" (must be unique)

3
Copy your Bot Token

BotFather will give you a token like: 123456789:ABCdefGHIjklmnoPQRstUVwxyz

Keep this safe! This is your BOT_TOKEN

Step 2: Get Your Chat ID

1
Search for your bot and start it

Open Telegram and search for your bot username

Tap START or send any message

2
Get your Chat ID

Visit this URL in your browser:

https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates

Replace YOUR_BOT_TOKEN with your actual token

3
Find your Chat ID in the response

Look for "chat":{"id":123456789}

Your Chat ID is the number: 123456789

Step 3: Configure Your Portfolio

1
Edit the tracker script

Open: /assets/js/visitor-tracker.js

2
Find the Telegram section and update:
const BOT_TOKEN = 'YOUR_TELEGRAM_BOT_TOKEN';

Replace with your actual token

3
Update Chat ID:
const CHAT_ID = 'YOUR_CHAT_ID';

Replace with your Chat ID

‚úÖ Done! You'll now receive Telegram notifications for each visitor

Discord Setup

‚úÖ Best for: Server notifications, team alerts

Step 1: Create a Discord Webhook

1
Open your Discord server

Right-click on a channel (or go to channel settings)

2
Go to Integrations ‚Üí Webhooks

Click "New Webhook"

3
Name it "Portfolio Visitor Alerts"

Click "Copy Webhook URL"

This is your WEBHOOK_URL

Step 2: Configure Your Portfolio

1
Edit the tracker script

Open: /assets/js/visitor-tracker.js

2
Find the sendToDiscord function and uncomment it
// sendToDiscord(data);

Remove the // to enable it

3
Update the Webhook URL:
const WEBHOOK_URL = 'YOUR_WEBHOOK_URL';

Paste your webhook URL here

‚úÖ Done! Discord messages will appear in your channel

Email Setup

‚úÖ Best for: Simple, no accounts needed

Option 1: Using Formspree (Free & Easy)

1
Visit formspree.io

Sign up for a free account

2
Create a new form

Get your form ID (looks like: abc123xyz)

3
Update the tracker script:
const FORMSPREE_ID = 'YOUR_FORMSPREE_ID';
4
Enable email notifications in Formspree settings

You'll get an email for each visitor!

Option 2: Using EmailJS

1
Visit emailjs.com

Sign up for free

2
Add your email service and get credentials

Service ID, Template ID, Public Key

3
Update the sendToEmail function

Use EmailJS library similar to what's already in your portfolio

‚úÖ Done! You'll receive emails for visitor alerts
